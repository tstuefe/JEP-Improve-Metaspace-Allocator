- The prototype is being developed in the jdk/sandbox repository in branch "stuefe-new-metaspace-branch": (http://hg.openjdk.java.net/jdk/sandbox)

- A current patch against jdk head (at the moment, jdk 15) can be found under: https://github.com/tstuefe/JEP-Improve-Metaspace-Allocator/blob/master/big.patch. This patch is regularly updated and rebased.

- Example programs to test the new allocator can be found under: https://github.com/tstuefe/JEP-Improve-Metaspace-Allocator/tree/master/examples. In particular, InterleavedLoaders has been used to measure both memory gains and - if used in unattended mode without wait times, performance.
